# Nitro Camera 🚀

Zero-overhead native bindings for Flutter. **Nitro** is the runtime layer of the **Nitrogen SDK** — it provides the base classes, annotations, and Dart-side runtime primitives that make type-safe, zero-copy FFI plugins possible on iOS and Android with zero method-channel overhead.

This package is the runtime dependency. Built explicitly for high-performance camera applications where every millisecond counts.

[![Nitro](https://img.shields.io/badge/Powered%20by-Nitro-cyan.svg)](https://github.com/mrousavy/nitro)
[![Flutter](https://img.shields.io/badge/Flutter-3.0+-blue.svg)](https://flutter.dev)

## ⚡ Why Nitro?

| Feature | Method Channel | FFI (manual) | **Nitro** |
| :--- | :---: | :---: | :---: |
| **Overhead per call** | ~0.3 ms | ~0 ms | **~0 ms** |
| **Type safety** | stringly-typed | manual | **generated, strict** |
| **Async support** | ✅ | manual isolates | **✅ generated** |
| **Streams** | ✅ slow | manual SendPort | **✅ zero-copy** |
| **Zero-copy buffers** | ❌ | manual | **✅ @HybridStruct** |
| **Code to write** | lots | enormous | **3 files** |

## 🌟 Vision Camera Features

Nitro Camera supports a **GPU-accelerated video pipeline** (OpenGL ES 2.0) similar to [Vision Camera](https://github.com/mrousavy/react-native-vision-camera) in React Native.

*   **Custom Shaders**: Apply real-time GLSL fragment shaders directly to the camera preview.
*   **Planar Frame Analysis**: Stream raw frames in **YUV_420_888** (Planar) for efficient QR, Barcode, and Face detection.
*   **Zero-Latency Filters**: Process frames at native speeds (60 FPS+) with zero CPU-to-GPU copies.

## 🛠️ Performance Benchmark

Because Nitro Camera avoids the asynchronous serialization of `MethodChannels`, it reduces the bridge overhead by up to **10x** compared to standard plugins. This is critical for high-frequency hardware adjustments and real-time computer vision.

## 📂 Project Structure

*   `android/`: Native Android implementation using `Camera2` and a custom **OpenGL Renderer**.
*   `ios/`: Native iOS implementation using `AVFoundation`.
*   `lib/`: Clean, strongly-typed Dart API generated by Nitro.
*   `example/`: A premium Flutter application showcasing glassmorphic UI and real-time vision diagnostics.

## 🚀 Getting Started

### 1. Add Dependency
Add `nitro_camera` to your `pubspec.yaml`:
```yaml
dependencies:
  nitro_camera:
    path: ../nitro_camera
```

### 2. Usage
```dart
// Launch the camera with GPU filtering
final textureId = await NitroCamera.instance.openCamera(config);

// Apply a real-time "Night Vision" shader
await NitroCamera.instance.setFilterShader(textureId, """
  #extension GL_OES_EGL_image_external : require
  precision mediump float;
  varying vec2 vTextureCoord;
  uniform samplerExternalOES sTexture;
  void main() {
      vec4 color = texture2D(sTexture, vTextureCoord);
      gl_FragColor = vec4(0.0, color.g * 1.5, 0.0, 1.0);
  }
""");

// Display the preview
Texture(textureId: textureId)
```
